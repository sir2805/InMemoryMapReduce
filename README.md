в виде: название_файла/host:portNo:название_файла_на_этой_ноде/host:portNo:название_файла_на_этой_ноде и так далее.

Для правильной работы программы последовательно запускается MapReduceNode, затем MapReduceMaster, затем MapReduceСlient и далее работа идет консолью MapReduceClient.
